# KandilliAPI
A simple API for getting data from [Kandilli observatory](http://www.koeri.boun.edu.tr/scripts/lst0.asp)
## Building

```bash
# install dependencies
npm i
# host for production
npm run start 
```

## Development

```bash
# install dependencies
npm i
# host local server
npm run dev
```
